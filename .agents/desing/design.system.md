# 🎨 Design System - Loyalty Web

## 🎯 Objetivo

Definir un sistema visual consistente para el frontend.

Debe ser:

- limpio
- moderno
- reutilizable
- alineado a dashboard SaaS

---

## 🎨 Estilo general

- Minimalista
- Espaciado amplio
- Bordes redondeados (lg)
- Sombras suaves

---

## 🎯 Colores

Primario:
- Azul (#2563eb)

Secundario:
- Gris oscuro (#111827)

Fondo:
- Gris claro (#f9fafb)

Estados:
- Success: verde
- Error: rojo
- Warning: amarillo

---

## 🔤 Tipografía

- Font: Inter / system
- Títulos: bold
- Texto: regular

---

## 📦 Componentes base

- Button
- Input
- Card
- Form
- Modal
- Alert

---

## 📏 Espaciado

Usar escala de Tailwind:

- p-2, p-4, p-6
- gap-4, gap-6

---

## 🧩 Reglas

- No estilos inline
- Usar clases reutilizables
- Componentizar UI
