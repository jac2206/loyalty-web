# 🎨 UI Instructions

## Reglas

- Usar Tailwind CSS
- Componentes reutilizables en shared/components
- No repetir estilos

---

## Formularios

- Inputs con label
- Mensajes de error claros
- Botón deshabilitado en loading

---

## Estados

Cada pantalla debe manejar:

- loading
- error
- empty

---

## UX

- Feedback inmediato
- Mensajes claros
- Navegación simple
