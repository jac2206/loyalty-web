# 🧠 Skill: UI Frontend

Especialización en construcción de interfaces modernas.

## 🎯 Objetivo

Construir interfaces:

- limpias
- reutilizables
- consistentes
- enfocadas en UX

---

## 📦 Patrones

### 1. Form Pattern

Siempre:

- label
- input
- error message
- button

---

### 2. Loading Pattern

- deshabilitar botón
- mostrar feedback

---

### 3. Error Pattern

- mensaje claro
- no técnico

---

### 4. Layout Pattern

- usar cards
- centrado para auth
- sidebar para dashboard

---

## 🎨 Integración con diseño

- usar Tailwind
- respetar design system
- no estilos inline

---

## ⚠️ Reglas

- no duplicar componentes
- no mezclar lógica con UI
- componentes pequeños y reutilizables
